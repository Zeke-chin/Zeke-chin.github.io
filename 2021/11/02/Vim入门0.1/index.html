前言

最近在学习linux时注意到了Vim这个大家都说很厉害的工具

其实在此之前学习生活中也是有注意到这个工具的但是只限于i,:w,:q这些基本命令

机缘巧合之下刷到一位大佬的视频引起了我学习vim的兴趣

写在前面

如上文所说

小z初始vim的时候 就是看了某个网上对于配文文件的命令vim config

然后进去之后就是一头雾水发现esc没用Contr+z/c全都都没用

不得已只能cd进目录用vscode进行修改 非常之麻烦

所以在教程开始之前我先教大家一下如何修改文件，保存文件，以及退出那我曾经视为洪水猛兽的vim命令行

常用动作motion

输入

  • i：在光标前插入

  • I：在行首插入

    • 👉点击查看示例👈

  • a：在光标后一位插入

  • A：在行尾插入

    • 👉点击查看示例👈

  • o：在下一行插入新行

  • O：在上一行插入新行

    • 👉点击查看示例👈

移动

  • 上下左右

    • 上

      • k————K

    • 下

      • j————J

    • 左

      • h————H

    • 右

      • l————L

        👉点击查看示例👈

  • gg/G：至文首/文尾

    • 👉点击查看示例👈

  • e/E：至词尾/连续单词词尾

    • 👉点击查看示例👈

  • b/B：至上个单词词尾/连续单词词尾

    • 👉点击查看示例👈

  • w/W:至下个单词词尾/词首

    • 👉点击查看示例👈

删除

  • J：合并两行，删除该行下一个↵

  • x：删除光标后一个字符

  • X：删除光标前一个字符**

    • 👉点击查看示例👈

  • d：删除

  • D：删除该行

    • 👉点击查看示例👈

撤销

  • u：撤销

  • U：撤销该行命令

    • 👉点击查看示例👈

拷贝

  • y：拷贝

  • Y：拷贝改行

    • 👉点击查看示例👈

黏贴

  • p：光标后

  • P：光标前

    • 👉点击查看示例👈

查找

  • f：在光标后查找并跳转

  • F：在光标前查找并跳转

    • 👉点击查看示例👈 ![](https://pic.imgdb.cn/item/61822e712ab3f51d9123a6cb.gif)

当一个Ubuntu的虚拟机或者服务器创建好之后应该做一写什么

使用SSH链接会比图像化界面让我更加舒服

使用ssh链接

在使用ssh链接之前 你应该确认你的linux是否安装了ssh服务

如果没有使用

apt-get install openssh-server来安装

再在本机使用ssh链接

更新应用

  • 获取最新的软件列表

    sudo apt update

  • 更新应用

​ sudo apt upgrade

安装配置我的zsh

    sudo apt install zsh

chocolatey卸载
chocolatey卸载相对来说要麻烦一些，但是还好一般不需要卸载。但是为了避免一些特殊情况，这里还是简单说明一下步骤。

chocolatey无法自动一键卸载，我们需要手动对其进行删除：

1.删除chocolatey的安装包，通常位于C:\ProgramData\chocolatey

2.删除环境变量中和chocolatey相关的内容，通常包含如下环境变量：

ChocolateyInstall
ChocolateyToolsLocation
ChocolateyLastPathUpdate
PATH (删除和chocolatey相关的环境变量)
chocolatey的使用
搜索
chocolatey可以直接通过命令行搜索支持的程序，但是更直观的方式是通过其官方网站进行搜索：https://chocolatey.org/packages，直接访问网址，在搜索框中输入关键字即可，比如我需要搜索一些文本编辑器，可以直接搜索note。

对note关键字进行搜索

找到notepad++
可以自行在搜索结果中寻找感兴趣的软件，找到后在软件的右下角可以看到安装命令，直接将命令复制到powershell中执行即可，例如要安装notepad++，可以直接复制命令：choco install notepadplusplus.install

安装中
其间会弹出提示询问是否运行脚本，输入y，回车即可。

输入“y”，确认安装
安装成功后，直接在命令行输入notepad++即可运行软件

运行notepad++
命令
chocolatey的操作都需要通过choco指令来运行，choco指令后可以跟随其他的子指令从而实现不同的功能。

choco list/search -l

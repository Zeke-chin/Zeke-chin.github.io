git命令

  • 创建仓库

    • 本地创建->git init
    • clone远程->git clone [url ]

  • Git 文件状态

    • Untracked ——未跟踪
      • 野生文件 可添加到Stage(暂存区)
        • 被标记git add [file]==>Untracked–>Staged
    • Unmodify ——stage内文件未修改
      • 标记文件 和Untracked一样
        • 被改动cont+s==>Untracked–>Modified
        • 被移除git rm [file]==>Untracked–>Untracked
    • Modified ——stage内文件已修改
      • 标记文件 和Untracked不
        • 被确实git add .==>Modified–>Staged
        • 被丢弃gie checkout [file]==>Modified–>Unmodify
    • Staged ——stage内文件
      • 暂存文件 等待被提交同步或取消暂存
        • 被提交git commit -m [message] ==>Staged–>Unmodify
        • 被取消git reset HEAD [file]==>Staged–>Modified

  • 查看文件状态

    • 查看指定文件git status [file]
    • 查看所有文件git status [file]
